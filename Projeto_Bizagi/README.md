Projeto Bizagi 2:

Um fazendeiro descobriu em suas terras uma fonte de água mineral puríssima e
com características que podem gerar um novo negócio. Apesar de toda animação,
iniciar a produção tem grandes dificuldades, pois o local onde se encontra esta
fonte é de difícil acesso e longe de onde o fazendeiro concentra outras atividades
de outros negócios que possui. Como ele sempre foi muito empreendedor e tem
recursos financeiros, ele decidiu arriscar um projeto que seja capaz de trazer a água
desta fonte e construir um local para processá-la, engarrafa-la e distribuí-la por
todo país.
Você é o engenheiro de computação que foi contratado para automatizar este
processo, devendo apresentar uma solução, utilizando para tal: Fluxograma de
processos, especificações técnicas como eletrônica (sensores), CLP,
microcontroladores, robótica, linguagens de programação etc.
